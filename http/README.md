## Task
Write class `MishaServer` with interface 
```javascript
class MishaServer {
    getWeather() {};
}
```
Method `getWeather` returns weather for Moscow.

For getting weather info you have to use public API, for example
https://rapidapi.com/community/api/open-weather-map/
